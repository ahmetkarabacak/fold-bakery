# Project Brief: Fold Bakery Landing Page

## Project Overview
A modern, high-end landing page for "Fold", an artisan bakery specializing in croissants, sourdough bread, gourmet sandwiches, and specialty coffee.

## Core Requirements

### Brand Identity
- **Name:** Fold (referencing the croissant lamination process)
- **Vibe:** Minimalist, warm, sophisticated
- **Focus:** Artisan quality, craftsmanship, premium experience

### Technical Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Icons:** Lucide React
- **Fonts:** 
  - Headings: Playfair Display (serif)
  - Body: Inter (sans-serif)

### Design System
- **Primary Colors:**
  - Buttery Yellow: `#F5E6C8`
  - Golden Brown: `#D4A017`
  - Dark Roast: `#3E2723`
  - Off-White: `#FAFAFA`
- **Visual Style:** Large imagery, generous whitespace, rounded corners

### Critical Requirements
1. **Mobile-First Design:** All CSS must start with mobile styles, using Tailwind breakpoints (`md:`, `lg:`) for larger screens
2. **Responsive Navigation:** Hamburger menu on mobile with slide-in drawer using Framer Motion
3. **Responsive Grids:** 1 column (mobile) → 2 columns (tablet) → 3-4 columns (desktop)
4. **Touch Targets:** Minimum 44px height for all interactive elements
5. **Accessibility:** Proper semantic HTML, readable text sizes on all devices

## Sections Required
1. Responsive Navbar (logo + navigation)
2. Hero Section (full-screen, animated entry)
3. Product Showcase (responsive grid: Croissants, Bread, Sandwiches, Coffee)
4. About Section (story about dough lamination)
5. Footer (stacked mobile, spread desktop)

## Success Criteria
- Fully responsive across all device sizes
- Smooth animations and transitions
- Fast loading and optimized performance
- Accessible and user-friendly
- Modern, premium aesthetic matching brand identity
